# WRO Future Engineers Randomizer

A web-based randomizer for the **WRO Future Engineers 2026** category, developed by **HyperLine Robotics**.

🔗 [https://wro-fe-randomizer.hyperlinerobotics.com/](https://wro-fe-randomizer.hyperlinerobotics.com/)

## Features

* Open Challenge randomizer
* Final Round randomizer
* Valid start position display
* Inner wall placement
* Cube and parking slot generation
* Built-in timer
* Desktop and mobile layout
* Randomize lock while timer is running

## How to Use

Open `index.html` in your browser.

No installation or backend is needed.

```text
index.html
css/styles.css
js/randomizer.js
assets/playfield.png
```

## Notes

This tool is made for practice and testing. Always check the latest official WRO rules before using any setup in an official context.
